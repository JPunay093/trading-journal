# Trading Journal

A secure, full-stack web application for tracking and analyzing your trading activities. Built with Next.js, React, TypeScript, and Supabase.

## Features

- 🔐 Secure user authentication with Supabase
- 📊 Real-time equity curve visualization
- 📈 Trade entry/exit tracking with P&L calculations
- 📅 Date filtering by presets (Today, This Week, This Month, All Time)
- 📥 CSV export functionality
- 📱 Responsive design with Tailwind CSS
- 🎯 Interactive date picker for easy date selection

## Tech Stack

- **Frontend:** Next.js 14, React 18, TypeScript
- **Styling:** Tailwind CSS
- **Database:** Supabase (PostgreSQL)
- **Authentication:** Supabase Auth
- **Charts:** Recharts
- **Forms:** React Hook Form
- **CSV:** PapaParse
- **Date Picker:** React DatePicker
- **Deployment:** Vercel

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn
- Supabase account

### Installation

1. Clone the repository:

```bash
git clone <your-repo-url>
cd trading-journal
```
